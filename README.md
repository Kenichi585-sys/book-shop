# Book Shop

TypeScript で作成された書籍ショップのコンソールアプリケーションです。お金をチャージして本を購入し、クーポンを使用して割引を受けることができます。

## 機能

- 💰 お金のチャージ
- 🛒 カート機能
- 📚 本の購入
- 📖 購入履歴の確認
- 🎫 クーポンの適用（10%OFF、500 円引き）

## セットアップ方法

### 前提条件

- Node.js (v18 以上)
- npm

### インストール

1. リポジトリをクローンします：

```bash
git clone <repository-url>
cd book-shop
```

2. 依存関係をインストールします：

```bash
npm install
```

### 実行方法

アプリケーションを実行するには以下のコマンドを使用します：

```bash
node main.ts
```

### 使用方法

アプリケーションを起動すると、以下のメニューが表示されます：

1. **お金をチャージする** - 残高にお金を追加
2. **カートを確認する** - 現在のカート内容と残高を表示
3. **商品を購入する** - 利用可能な本から購入
4. **購入履歴を確認する** - 過去の購入履歴を表示
5. **クーポンを適用する** - 割引クーポンを適用
6. **終了する** - アプリケーションを終了

### 利用可能なクーポン

- `TS10`: 10%OFF
- `SAVE500`: 500 円引き
